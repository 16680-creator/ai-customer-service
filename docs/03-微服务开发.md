所有接口统一使用 `Result<T>` 包装返回值：

```java
// 成功返回
return Result.success();                    // {"code":200,"message":"操作成功","data":null,"timestamp":...}
return Result.success(user);                // {"code":200,"message":"操作成功","data":{...},"timestamp":...}

// 失败返回
return Result.fail();                       // {"code":500,"message":"操作失败","data":null,"timestamp":...}
return Result.fail("用户名已存在");           // 自定义消息
return Result.fail(ResultCode.USER_ALREADY_EXISTS);  // 使用错误码
return Result.fail(ResultCode.CHAT_AI_SERVICE_UNAVAILABLE, "AI服务调用失败: timeout");
```

```java
    private String message; // 提示信息
    private T data;         // 响应数据
    private long timestamp; // 时间戳
}
```

## 4. 用户服务开发指南

### 4.1 注册/登录流程

**注册流程**（`UserServiceImpl.register()`）：

```
1. 接收 User 对象（username, password, nickname 等）
2. 检查用户名是否已存在 → 存在则抛出 BusinessException(USER_ALREADY_EXISTS)
3. 使用 BCrypt 加密密码
4. 设置默认状态(1)和角色(user)
5. 插入数据库
6. 返回 Result.success()
```

**登录流程**（`UserServiceImpl.login()`）：

```
1. 接收 username 和 password
2. 查询用户 → 不存在则抛出 BusinessException(USER_NOT_FOUND)
3. BCrypt 校验密码 → 错误则抛出 BusinessException(USER_PASSWORD_ERROR)
4. 检查用户状态 → 禁用则抛出 BusinessException(USER_ACCOUNT_DISABLED)
5. 生成 JWT Token（包含 userId, username, role）
6. 返回 Result.success(token)
```

### 4.2 JWT 鉴权流程

网关 `AuthFilter` 统一处理鉴权：

```
1. 请求进入网关
2. 检查路径是否在白名单（/api/user/login, /api/user/register 等）
3. 白名单直接放行
4. 非白名单：从 Authorization 头提取 Bearer Token
5. JwtUtil.validateToken() 校验 Token
6. 解析 Token 获取 userId 和 username
7. 将用户信息写入请求头（X-User-Id, X-User-Name）透传至下游
8. 下游服务通过 @RequestHeader 获取用户信息
```

```xml
<dependency>
    <groupId>io.minio</groupId>
    <artifactId>minio</artifactId>

</dependency>
```

**2. 配置 MinIO**（`application.yml`）：

```yaml
minio:
  endpoint: http://127.0.0.1:9000
  access-key: minioadmin
  secret-key: minioadmin
  bucket: aics-knowledge
```

**3. MinIO 配置类**：

```java
@Configuration
public class MinioConfig {

    @Value("${minio.endpoint}")
    private String endpoint;

    @Value("${minio.access-key}")
    private String accessKey;

    @Value("${minio.secret-key}")
    private String secretKey;

    @Bean
    public MinioClient minioClient() {
        return MinioClient.builder()
                .endpoint(endpoint)
                .credentials(accessKey, secretKey)
                .build();
    }
}
```

**4. 文件上传接口**：

```java
@PostMapping("/upload")
public Result<String> upload(@RequestParam("file") MultipartFile file) {
    try {
        String objectName = UUID.randomUUID() + "_" + file.getOriginalFilename();
        minioClient.putObject(PutObjectArgs.builder()
                .bucket(bucketName)
                .object(objectName)
                .stream(file.getInputStream(), file.getSize(), -1)
                .contentType(file.getContentType())
                .build());
        String url = endpoint + "/" + bucketName + "/" + objectName;
        return Result.success(url);
    } catch (Exception e) {
        throw new BusinessException(ResultCode.KNOWLEDGE_UPLOAD_FAIL, "文件上传失败: " + e.getMessage());
    }
}
```

### 6.2 RAG 流程

当前 RAG 实现流程（`ChatServiceImpl.chatWithRag()`）：

```
1. 接收用户消息和知识库标识
2. 构建包含知识库上下文的提示词
3. 调用 ChatClient 生成回答
4. 返回 AI 回复
```

**DocumentLoader** 支持加载 PDF 和文本文件：

```java
// 加载 PDF 文档
List<Document> pdfDocs = documentLoader.loadPdf(new FileSystemResource("docs/faq.pdf"));

// 加载文本文档
List<Document> textDocs = documentLoader.loadText(new FileSystemResource("docs/policy.txt"));
```

**完整 RAG 流程（生产环境推荐）**：

```
1. 文档加载：DocumentLoader.loadPdf() / loadText()
2. 文档分片：TextSplitter.split()
3. 向量化：EmbeddingModel.embed()
4. 存储：VectorStore.add()
5. 检索：VectorStore.similaritySearch()
6. 生成：ChatClient.prompt().user(question).call()
```

### 6.3 流式对话

`ChatServiceImpl.chatStream()` 提供流式对话入口，实际流式输出通过 SSE 或 WebSocket 推送：

```java
// SSE 流式输出示例
@GetMapping(value = "/chat/stream", produces = MediaType.TEXT_EVENT_STREAM_VALUE)
public Flux<String> chatStream(@RequestParam String sessionId,
                                @RequestParam String message) {
    return chatModel.stream(message)
            .map(chatResponse -> {
                String content = chatResponse.getResult().getOutput().getContent();
                return content != null ? content : "";
            });
}
```

### 8.2 生产者示例

```java
@Service
@RequiredArgsConstructor
@Slf4j
public class MessageProducer {

    private final RocketMQTemplate rocketMQTemplate;

    /**
     * 发送知识库处理消息
     */
    public void sendKnowledgeProcessMessage(Long documentId) {
        String destination = "knowledge-process";
        String message = String.valueOf(documentId);
        rocketMQTemplate.convertAndSend(destination, message);
        log.info("发送知识库处理消息: documentId={}", documentId);
    }

    /**
     * 发送通知消息
     */
    public void sendNotifyMessage(String userId, String content) {
        NotifyMessage msg = new NotifyMessage(userId, content);
        rocketMQTemplate.convertAndSend("notify-message", msg);
        log.info("发送通知消息: userId={}", userId);
    }
}
```

### 9.3 WebSocket 端点

```java
@Component
@ServerEndpoint("/ws/notify/{userId}")
@Slf4j
public class NotifyWebSocket {

    private static final ConcurrentHashMap<String, Session> SESSIONS = new ConcurrentHashMap<>();

    @OnOpen
    public void onOpen(@PathParam("userId") String userId, Session session) {
        log.info("WebSocket连接建立: userId={}", userId);
        SESSIONS.put(userId, session);
    }

    @OnClose
    public void onClose(@PathParam("userId") String userId) {
        log.info("WebSocket连接关闭: userId={}", userId);
        SESSIONS.remove(userId);
    }

    @OnMessage
    public void onMessage(@PathParam("userId") String userId, String message) {
        log.info("收到WebSocket消息: userId={}, message={}", userId, message);
    }

    /**
     * 向指定用户推送消息
     */
    public static void sendMessage(String userId, String message) {
        Session session = SESSIONS.get(userId);
        if (session != null && session.isOpen()) {
            session.getAsyncRemote().sendText(message);
        }
    }

    /**
     * 广播消息
     */
    public static void broadcast(String message) {
        SESSIONS.values().forEach(session -> {
            if (session.isOpen()) {
                session.getAsyncRemote().sendText(message);
            }
        });
    }
}
```

### 9.4 消息推送集成

```java
@Service
@RequiredArgsConstructor
@Slf4j
public class NotifyServiceImpl implements NotifyService {

    private final RocketMQTemplate rocketMQTemplate;

    /**
     * 发送通知（通过 RocketMQ 异步推送）
     */
    @Override
    public Result<Void> sendNotify(String userId, String content) {
        NotifyMessage message = new NotifyMessage(userId, content, LocalDateTime.now());
        rocketMQTemplate.convertAndSend("notify-message", message);
        return Result.success();
    }
}

@Component
@RocketMQMessageListener(
    topic = "notify-message",
    consumerGroup = "aics-notify-consumer-group"
)
@Slf4j
public class NotifyConsumer implements RocketMQListener<NotifyMessage> {

    @Override
    public void onMessage(NotifyMessage message) {
        log.info("收到通知消息: userId={}", message.getUserId());
        // 通过 WebSocket 推送给用户
        NotifyWebSocket.sendMessage(message.getUserId(), message.getContent());
    }
}
```
