## 1. Spring AI 概述与架构

### 1.1 Spring AI 是什么

Spring AI 是 Spring 官方推出的 AI 模型集成框架，提供统一的 API 抽象层，支持多种大语言模型（OpenAI、Ollama、Azure OpenAI、HuggingFace 等）。项目使用 Spring AI 1.0.0 版本。

### 1.2 项目中的 AI 架构

```
用户请求
    │
    ▼
┌──────────────┐     ┌──────────────────┐
│  ChatController │───▶│  ChatServiceImpl  │
└──────────────┘     └──────────────────┘
                              │
                    ┌─────────┼─────────┐
                    ▼         ▼         ▼
              ┌──────────┐ ┌───────┐ ┌────────────┐
              │ ChatClient│ │ RAG   │ │ ChatMemory │
              └──────────┘ └───────┘ └────────────┘
                    │         │
                    ▼         ▼
              ┌──────────┐ ┌──────────────┐
              │ LLM Model │ │ VectorStore  │
              └──────────┘ └──────────────┘
                    │         │
              ┌─────┴─────┐  ▼
              │           │ ┌──────────────┐
         ┌────┴───┐ ┌────┴─┐│ EmbeddingModel│
         │ OpenAI │ │Ollama│└──────────────┘
         └────────┘ └──────┘
```

## 2. Ollama 本地模型安装

### 2.1 安装 Ollama

```bash
# macOS
brew install ollama

# Linux
curl -fsSL https://ollama.com/install.sh | sh

# Windows
# 下载安装包：https://ollama.com/download/windows
```

### 2.2 启动 Ollama 服务

```bash
# 启动 Ollama（默认端口 11434）
ollama serve

# 验证服务
curl http://localhost:11434/api/tags
```

### 2.3 拉取推荐模型

```bash
# 拉取对话模型（推荐 qwen2.5:7b，中文能力强）
ollama pull qwen2.5:7b

# 拉取 Embedding 模型（用于 RAG 向量化）
ollama pull nomic-embed-text

# 其他可选对话模型
ollama pull llama3:8b          # Meta Llama 3 8B
ollama pull mistral:7b         # Mistral 7B
ollama pull glm4:9b            # 智谱 GLM-4 9B

# 查看已安装模型
ollama list

# 测试模型
ollama run qwen2.5:7b "你好，请介绍一下自己"
```

### 2.4 配置项目使用 Ollama

修改 `ai-cs-chat/src/main/resources/application.yml`：

```yaml
spring:
  ai:
    ollama:
      base-url: http://localhost:11434
      chat:
        model: qwen2.5:7b
        options:
          temperature: 0.7
          num-predict: 2048
      embedding:
        model: nomic-embed-text
        options:
          dimension: 768
```

同时需要将依赖从 `spring-ai-openai-spring-boot-starter` 切换为 `spring-ai-ollama-spring-boot-starter`：

```xml
<!-- 替换 OpenAI starter 为 Ollama starter -->
<dependency>
    <groupId>org.springframework.ai</groupId>
    <artifactId>spring-ai-ollama-spring-boot-starter</artifactId>
</dependency>
```

## 4. RAG 实现详解

### 4.1 RAG 流程概述

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│  文档加载    │───▶│  文档分片    │───▶│  向量化存储  │
│ DocumentLoader│   │ TextSplitter │   │ VectorStore  │
└─────────────┘    └─────────────┘    └─────────────┘
                                            │
┌─────────────┐    ┌─────────────┐          │
│  AI 生成    │◀───│  检索增强    │◀─────────┘
│ ChatClient   │    │ SimilaritySearch│
└─────────────┘    └─────────────┘
```

### 4.2 文档加载

项目 `DocumentLoader.java` 已实现 PDF 和文本文件加载：

```java
@Component
public class DocumentLoader {

    /**
     * 加载 PDF 文档
     */
    public List<Document> loadPdf(Resource resource) {
        PagePdfDocumentReader pdfReader = new PagePdfDocumentReader(
                resource,
                PdfDocumentReaderConfig.builder()
                        .withPageTopMargin(0)
                        .withPageBottomMargin(0)
                        .withPageExtractedTextFormatter(extractedTextFormatter ->
                                extractedTextFormatter.withLeftAlignment(true))
                        .withPagesPerDocument(1)
                        .build()
        );
        return pdfReader.get();
    }

    /**
     * 加载文本文档
     */
    public List<Document> loadText(Resource resource) {
        TextReader textReader = new TextReader(resource);
        return textReader.get();
    }
}
```

支持的文档类型扩展：

```java
// 加载 Markdown 文档（需添加依赖）
```

### 4.3 文档分片

```java
import org.springframework.ai.transformer.splitter.TokenTextSplitter;

@Component
public class DocumentSplitter {

    /**
     * 将文档分片
     * @param documents 原始文档列表
     * @return 分片后的文档列表
     */
    public List<Document> split(List<Document> documents) {
        TokenTextSplitter splitter = new TokenTextSplitter(
                800,    // 默认分片大小（token 数）
                200,    // 重叠大小
                5,      // 最小分片长度
                10000,  // 最大分片长度
                true    // 保留分隔符
        );
        return splitter.apply(documents);
    }
}
```

### 4.4 向量化与存储

```java
@Configuration
public class RagConfig {

    @Bean
    public VectorStore vectorStore(EmbeddingModel embeddingModel) {
        return new SimpleVectorStore(embeddingModel);
    }
}

@Service
@RequiredArgsConstructor
public class KnowledgeIndexService {

    private final DocumentLoader documentLoader;
    private final DocumentSplitter documentSplitter;
    private final VectorStore vectorStore;

    /**
     * 索引文档到向量库
     */
    public void indexDocument(Resource resource, String docType) {
        // 1. 加载文档
        List<Document> documents = switch (docType) {
            case "pdf" -> documentLoader.loadPdf(resource);
            case "txt" -> documentLoader.loadText(resource);
            default -> throw new BusinessException(ResultCode.KNOWLEDGE_PARSE_FAIL, "不支持的文档类型: " + docType);
        };

        // 2. 文档分片
        List<Document> chunks = documentSplitter.split(documents);

        // 3. 向量化并存储
        vectorStore.add(chunks);
    }
}
```

### 4.5 检索增强生成

```java
@Service
@RequiredArgsConstructor
public class ChatServiceImpl implements ChatService {

    private final ChatClient chatClient;
    private final VectorStore vectorStore;

    /**
     * RAG 对话（完整实现）
     */
    @Override
    public Result<String> chatWithRag(String sessionId, String message, String knowledgeBase) {
        try {
            // 1. 从向量库检索相关文档
            List<Document> relevantDocs = vectorStore.similaritySearch(
                    SearchRequest.query(message).withTopK(5).withSimilarityThreshold(0.7)
            );

            // 2. 构建上下文
            String context = relevantDocs.stream()
                    .map(Document::getContent)
                    .collect(Collectors.joining("\n\n"));

            // 3. 构建增强提示词
            String ragPrompt = String.format(
                    "请基于以下参考资料回答用户问题。如果参考资料中没有相关信息，请如实告知。\n\n" +
                    "参考资料：\n%s\n\n" +
                    "用户问题：%s",
                    context, message
            );

            // 4. 调用 AI 模型生成回答
            String response = chatClient.prompt()
                    .user(ragPrompt)
                    .call()
                    .content();

            return Result.success(response);
        } catch (Exception e) {
            throw new BusinessException(ResultCode.CHAT_AI_SERVICE_UNAVAILABLE, "AI服务调用失败: " + e.getMessage());
        }
    }
}
```

```java
public class VectorStoreConfig {

    private static final String VECTOR_STORE_FILE = "vector-store.json";

    @Bean
    public VectorStore vectorStore(EmbeddingModel embeddingModel) {
        SimpleVectorStore vectorStore = new SimpleVectorStore(embeddingModel);

        // 加载已有数据

            vectorStore.load(vectorStoreFile);

        return vectorStore;

        vectorStore.save(new File(VECTOR_STORE_FILE));
    }
}
```

### 6.3 使用 Embedding 模型

```java
@Service
@RequiredArgsConstructor
public class EmbeddingService {

    private final EmbeddingModel embeddingModel;

    /**
     * 计算文本向量
     */
    public float[] embed(String text) {
        return embeddingModel.embed(text);
    }

    /**
     * 计算文本相似度
     */
    public double similarity(String text1, String text2) {
        float[] embedding1 = embeddingModel.embed(text1);
        float[] embedding2 = embeddingModel.embed(text2);
        return cosineSimilarity(embedding1, embedding2);
    }

    private double cosineSimilarity(float[] a, float[] b) {
        double dotProduct = 0.0, normA = 0.0, normB = 0.0;
        for (int i = 0; i < a.length; i++) {
            dotProduct += a[i] * b[i];
            normA += a[i] * a[i];
            normB += b[i] * b[i];
        }
        return dotProduct / (Math.sqrt(normA) * Math.sqrt(normB));
    }
}
```

### 7.3 Redis 持久化 ChatMemory（生产环境）

```java
@Configuration
public class ChatMemoryConfig {

    @Bean
    public ChatMemory chatMemory(RedisTemplate<String, Object> redisTemplate) {
        return new RedisChatMemory(redisTemplate);
    }
}

/**
 * Redis 实现的 ChatMemory
 */
public class RedisChatMemory implements ChatMemory {

    private static final String KEY_PREFIX = "aics:chat:memory:";
    private static final Duration TTL = Duration.ofHours(24);

    private final RedisTemplate<String, Object> redisTemplate;

    public RedisChatMemory(RedisTemplate<String, Object> redisTemplate) {
        this.redisTemplate = redisTemplate;
    }

    @Override
    public void add(String conversationId, List<Message> messages) {
        String key = KEY_PREFIX + conversationId;
        messages.forEach(msg -> {
            redisTemplate.opsForList().rightPush(key, msg);
        });
        redisTemplate.expire(key, TTL);
    }

    @Override
    public List<Message> get(String conversationId, int lastN) {
        String key = KEY_PREFIX + conversationId;
        Long size = redisTemplate.opsForList().size(key);
        if (size == null || size == 0) {
            return List.of();
        }
        long start = Math.max(0, size - lastN);
        List<Object> raw = redisTemplate.opsForList().range(key, start, size - 1);
        if (raw == null) {
            return List.of();
        }
        return raw.stream()
                .filter(obj -> obj instanceof Message)
                .map(obj -> (Message) obj)
                .toList();
    }

    @Override
    public void clear(String conversationId) {
        redisTemplate.delete(KEY_PREFIX + conversationId);
    }
}
```

## 8. Function Calling 示例

### 8.1 定义 Function

Spring AI 支持将 Java 方法注册为 AI 可调用的 Function：

```java
@Configuration
public class AiFunctions {

    /**
     * 查询知识库函数
     */
    @Bean
    @Description("根据关键词搜索知识库，返回相关知识文档")
    public Function<SearchRequest, SearchResponse> searchKnowledge(
            KnowledgeService knowledgeService) {
        return request -> {
            var page = knowledgeService.listDocuments(
                    request.page(), request.pageSize(), request.keyword());
            return new SearchResponse(page);
        };
    }

    /**
     * 查询用户信息函数
     */
    @Bean
    @Description("根据用户ID查询用户信息")
    public Function<UserQueryRequest, UserQueryResponse> queryUser(
            UserService userService) {
        return request -> {
            var result = userService.getUserById(request.userId());
            return new UserQueryResponse(result);
        };
    }

    public record SearchRequest(String keyword, int page, int pageSize) {}
    public record SearchResponse(Object data) {}
    public record UserQueryRequest(Long userId) {}
    public record UserQueryResponse(Object data) {}
}
```

### 8.2 在 ChatClient 中启用 Function Calling

```java
@Configuration
public class SpringAiConfig {

    @Bean
    public ChatClient chatClient(OpenAiChatModel chatModel) {
        return ChatClient.builder(chatModel)
                .defaultSystem("你是一个专业的AI客服助手。你可以搜索知识库来回答用户问题。")
                .defaultFunctions("searchKnowledge", "queryUser")
                .build();
    }
}
```

### 8.3 运行时动态调用 Function

```java
@RestController
@RequestMapping("/chat")
@RequiredArgsConstructor
public class ChatController {

    private final ChatClient chatClient;

    @PostMapping("/function-call")
    public Result<String> chatWithFunction(@RequestParam String message) {
        String response = chatClient.prompt()
                .user(message)
                .functions("searchKnowledge")
                .call()
                .content();
        return Result.success(response);
    }
}
```

### 8.4 Function Calling 工作流程

```
1. 用户发送消息："退款政策是什么？"
2. AI 模型判断需要调用 searchKnowledge 函数
3. Spring AI 自动调用 searchKnowledge("退款政策", 1, 10)
4. 将函数返回结果注入到对话上下文
5. AI 模型基于函数结果生成最终回答
6. 返回给用户："根据知识库信息，购买后7天内可申请无理由退款..."
```

## 9. 性能优化建议

### 9.1 模型调用优化

| 优化项 | 说明 | 配置示例 |
|--------|------|---------|
| 降低 temperature | 客服场景建议 0.3-0.5，减少随机性 | `temperature: 0.3` |
| 限制 max-tokens | 控制回复长度，降低成本 | `max-tokens: 1024` |
| 设置超时 | 避免长时间等待 | `spring.ai.openai.chat.options.timeout=30s` |
| 流式输出 | 首字响应更快，提升用户体验 | 使用 `chatModel.stream()` |

### 9.2 RAG 优化

| 优化项 | 说明 |
|--------|------|
| 分片大小 | 建议 500-1000 token，过小丢失上下文，过大检索不精确 |
| 重叠大小 | 建议 100-200 token，避免关键信息被截断 |
| Top-K | 建议 3-5，过多引入噪声，过少遗漏信息 |
| 相似度阈值 | 建议 0.7-0.8，过滤低质量检索结果 |
| 混合检索 | 结合关键词检索和向量检索，提高召回率 |

```java
public class AiRateLimiter {

    private final Semaphore semaphore = new Semaphore(10);

    public <T> T executeWithLimit(Supplier<T> supplier) {
        try {
            semaphore.acquire();
            return supplier.get();
        } catch (InterruptedException e) {
            Thread.currentThread().interrupt();
            throw new BusinessException(ResultCode.CHAT_AI_SERVICE_UNAVAILABLE, "AI服务繁忙，请稍后重试");
        } finally {
            semaphore.release();
        }
    }
}
```

### 9.5 监控指标

```java
@Component
@RequiredArgsConstructor
@Slf4j
public class AiMetrics {

    private final AtomicLong totalRequests = new AtomicLong(0);
    private final AtomicLong successRequests = new AtomicLong(0);
    private final AtomicLong failedRequests = new AtomicLong(0);
    private final AtomicLong totalTokens = new AtomicLong(0);

    public void recordRequest(boolean success, long tokens, long durationMs) {
        totalRequests.incrementAndGet();
        if (success) {
            successRequests.incrementAndGet();
        } else {
            failedRequests.incrementAndGet();
        }
        totalTokens.addAndGet(tokens);
        log.info("AI调用统计 - 总请求: {}, 成功: {}, 失败: {}, 总Token: {}, 耗时: {}ms",
                totalRequests.get(), successRequests.get(), failedRequests.get(),
                totalTokens.get(), durationMs);
    }
}
```
